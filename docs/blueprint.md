# **App Name**: CognitoNote

## Core Features:

- Email/Password Authentication: Secure user authentication using Firebase Auth with email and password.
- Workspace Management: Create, list, switch, and delete workspaces to organize notes.
- Note Creation and Editing: Create, edit, and delete notes within workspaces, with auto-saving functionality.
- Tag Management: Add, remove, and filter notes by user-created tags to enhance organization.
- AI Summarization: Summarize notes using Google Gemini 2.0 Flash to provide concise summaries.
- AI Task Extraction: Tool: Extract action items from notes using Google Gemini 2.0 Flash to identify tasks.
- AI Tag Suggestion: Suggest tags for notes using Google Gemini 2.0 Flash to improve note organization.

## Style Guidelines:

- Primary color: Blue (#3B82F6) for trust and professionalism.
- Background color: Dark gray (#0D0D0D) to reduce eye strain and provide a modern feel.
- Accent color: Purple (#8B5CF6) to highlight AI-specific features and indicators.
- Display font: 'Space Grotesk' (sans-serif) for headings, for a modern, attention-grabbing feel.
- Body font: 'Inter' (sans-serif) for body and UI text, ensuring high readability.
- Monospace font: 'JetBrains Mono' (monospace) for code-like note content appearance.
- Lucide React icons, sized at 20px for UI elements and 24px for headers, with colors inheriting from parent text.
- Mobile-first responsive design with a single-column layout on mobile, expanding to a 2-3 column layout on larger screens.
- Subtle animations for interactive feedback, such as hover states and loading indicators.
